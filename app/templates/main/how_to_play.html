How to Play KenTrivia

For Players

  1. Get the game PIN from your host
  2. Enter the PIN and choose a nickname on the home page
  3. Wait in the lobby for the host to start the game
  4. Read each question carefully and select your answer
  5. Answer quickly! Faster answers earn more points
  6. Keep your answer streak going for bonus points
  7. Check the leaderboard between questions

For Hosts

  1. Create an account or sign in
  2. Create a new game from your dashboard
  3. Add questions manually or import from OpenTDB
  4. Share the game PIN with your players
  5. Wait for players to join in the lobby
  6. Start the game when ready
  7. Control the game flow and monitor progress

Scoring System

Base Points

  • Each question is worth up to 1000 points
  • Points decrease the longer you take to answer
  • Wrong answers receive 0 points

Bonus Points

  • Answer Streak: +10% per correct answer
  • Maximum streak bonus: +50%
  • Streak resets on wrong answers
